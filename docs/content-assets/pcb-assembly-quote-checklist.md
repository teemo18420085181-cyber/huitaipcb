## What files to prepare

- Gerber files and drill files
- BOM with manufacturer part numbers, quantity, designators, and approved alternates
- Pick-and-place file
- Assembly drawing with polarity / orientation notes
- Test instruction or acceptance criteria
- Sample photos (top and bottom)

## Minimum files for a preliminary review

- Sample photos of the board
- A partial BOM or component list
- An older Gerber, if available
- Board markings / silkscreen reference
- Quantity and target application

## Best files for an accurate quotation

- Production Gerber + drill files
- Clean BOM with MPNs and approved alternates
- Pick-and-place data
- Assembly drawing with polarity notes
- Quantity, target lead time, and testing requirements

## BOM risk checklist (mark these in your BOM)

- Critical ICs
- Connectors
- Components that require an exact brand
- Pre-programmed parts
- Known long-lead-time parts
- Obsolete parts

## Testing requirement checklist

- Visual inspection
- AOI inspection (where applicable)
- Polarity check
- Power-on test (if a test method is provided)
- Functional test based on buyer instructions
