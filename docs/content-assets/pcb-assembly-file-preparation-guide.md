## Quick Answer

To get an accurate PCB assembly quote, you usually need Gerber files, a BOM with manufacturer part numbers, pick-and-place data, an assembly drawing, quantity, and any testing requirements. Each file answers a different question for the assembler — fabrication, sourcing, placement, orientation, and test. If your files are incomplete, you can still start: send a partial BOM, an older Gerber, board markings, or sample photos, and an engineering review will tell you what is missing before quotation.

## Required files and what each one is for

| File | What it's used for |
| --- | --- |
| Gerber files | PCB fabrication and stack-up review |
| BOM | Component sourcing and cost review |
| Pick-and-place file | SMT placement setup |
| Assembly drawing | Polarity and orientation confirmation |
| Test instruction | Functional test planning |
| Sample photos | Useful when original files are incomplete |

## Minimum files for a preliminary quote

You do not need a perfect data package to start. For a preliminary review, the following is often enough:

- Sample photos of the board (top and bottom)
- A partial BOM or component list
- An older Gerber, if available
- Board markings or silkscreen reference
- Quantity
- Target application

## Best files for an accurate quote

For a firm, accurate quote with fewer revisions, send a complete set: production Gerber + drill files, a clean BOM with manufacturer part numbers and approved alternates, pick-and-place data, an assembly drawing with polarity notes, quantity, target lead time, and testing requirements.

## Common mistakes that delay quotation

- Missing BOM manufacturer part numbers
- Unclear component packages
- No quantity stated
- No testing requirement
- Unclear lead-time expectation
- Missing polarity or orientation notes

## Conclusion

For overseas buyers, the fastest way to get an accurate PCB assembly quote is to send Gerber files, BOM, quantity, lead time target, and testing requirements together. If some files are missing, Huitai Electronics can first review the available files or sample photos.
